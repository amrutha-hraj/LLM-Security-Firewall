# LLM Security Firewall Against Prompt Injection Attacks

An AI-powered cybersecurity firewall designed to detect and mitigate prompt injection and jailbreak attacks against Large Language Models using machine learning, semantic analysis, and real-time threat monitoring.

## Features
- Prompt Injection Detection
- Jailbreak Prevention
- Semantic Threat Analysis
- Risk Scoring
- Real-Time Monitoring
- LLM Protection Layer

- FastAPI
- Streamlit
- MongoDB
- Transformers
